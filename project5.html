Pokemon App - Figma

Project Overview

The Pokémon App is a simple mobile application developed using React Native and Expo, allowing users to view Pokémon cards, add new Pokémon to their collection, and edit details for existing ones. The app provides a user-friendly interface to manage a personalized collection of Pokémon in a clean and intuitive design.

Project Details

Duration: November 13th 2024 - November 20th 2024

Type: Mobile App Development Mini Project

Key Achievements

  • Developed a functional app featuring Pokémon cards with details like name, type, and stats
  • Implemented pages for adding and editing Pokémon, giving users control over their collection
  • Gained hands-on experience in React Native for building mobile applications
  • Enhanced skills in managing forms and data with React Native components

Technologies Used

WebStorm React Native Expo JavaScript
